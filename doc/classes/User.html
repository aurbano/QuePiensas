    User class, generate a User object for every user you want to interact with

    Sends private message to user, from current user

    sendPM(int $to, string $msg, int $thread, int $ident, int $com) : int

    All variables are checked within this function STATUS GUIDE 0 -> Just sent, unread 1 -> Read 2 -> Deleted by sender, unread 3 -> Deleted by sender, read 4 -> Deleted by receiver IDENT GUIDE # -> To Current User 0 -> Public Public 1 -> Private Public 2 -> Public Private 3 -> Private Private

    Parameters

    $to

    int

    User ID of receiver

    $msg

    string

    Message content, no need to clean it

    $thread

    int

    Thread ID, or 0 if PM is not a reply

    $ident

    int

    Check description above

    $com

    int

    If the message is in reply to a comment, the ID of the comment

    Returns

    intID of the sent PM or false if something failed
